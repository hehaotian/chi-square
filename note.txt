The command line of chi_feature_selection.sh is like this: 
chi_feature_selection <train_vector> <test_vector> <chi_thres> <feature_chi_file> <train_vector_out> <test_vector_out> 


train_vector: training vectors file
test_vector: testing vectors file
chi_thres: threshold of chi square, features below this threshold will not be selected
feature_chi_file: each feature and its chi square value
train_vector_out: training vectors file after feature selection
test_vector_out: testing vectors file after feature selection
